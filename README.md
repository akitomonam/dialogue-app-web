# 対話システム(WEB)
対話システム用のwebアプリです。  
WebSocketを用いてサーバーと通信を行います。
## 使い方
websocketのインストール
```
pip install websockets
```
サーバープログラム起動
```
python3 websocketServer.py
```
index.htmlをブラウザで開く
